1. Create a GWT application to develop GUI for Combo box with all cities. On selection of each City, populate all the places into a list using data from google places https://developers.google.com/places/web-service/search. 

2. Use Jetty server to run the GWT application.

3. Write a RPC call from client to talk to Service layer. A service class which is used to communicate with DAO layer. 

4. Implement the Dependency Injection to get the Service, DAO objects using GWT Guice.

5. DAO class to communicate with DB to perform actual CURD operations using JPA to save/update/read/delete the results from or to Database.

6. Use PostgreSQL to store the data such as all the cities & corresponding places.
